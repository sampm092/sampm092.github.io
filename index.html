MyBookList

This program is a simple application that I developed as an implementation of my study of the Laravel framework. The program features user login and registration, where created accounts can be used to manually record a list of books read by the user. The project could be improved by retrieving book data using API.

Features:

  • About
  • Homepage
  • Book Input
  • Book Details
  • User Login and Registration
  • The account homepage displays the books that have been entered
  • Account Settings: Change name, profile photo, delete account, and account details
  • Admin Page: Check the user list, delete users, change user passwords, and add admins

Program ini merupakan aplikasi sederhana yang dikembangkan sebagai bagian dari implementasi pembelajaran saya tentang framework Laravel. Aplikasi ini dilengkapi dengan fitur autentikasi pengguna, yaitu login dan registrasi, yang memungkinkan pengguna untuk mencatat dan mengelola daftar buku yang telah dibaca. Projek dapat ditingkatkan jika mengambil data buku menggunakan API.

Fitur:

  • About
  • Homepage
  • Input buku
  • Detail buku
  • Login dan Registrasi pengguna
  • Beranda akun menampilkan buku-buku yang telah di-input
  • Pengaturan akun: ubah nama, foto profil, hapus akun, dan detail akun
  • Halaman admin: cek daftar user, hapus user, ganti password user, dan menambah admin
